# 🤖 AI Game App

A simple web-based AI Game App built using *HTML, CSS, and JavaScript*.
This project runs fully online (GitHub Pages) with *no downloads or external libraries*.

---

## 🎮 Games Included

### 1️⃣ Tic-Tac-Toe
- Hard AI but *winnable*
- Player vs Computer
- Instant reset

### 2️⃣ Chess (Simple AI)
- Click to move pieces
- AI responds automatically
- Smart but beatable
- Beginner-friendly logic

### 3️⃣ Math Quiz (7–9 Standard)
- Easy + Hard math questions
- Instant feedback
- Continuous questions

### 4️⃣ Guess My Number
- Guess a number between 1 and 100
- Hints: Too High / Too Low
- Fully working logic
